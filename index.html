Projeto Finalizado

Projeto finalizado, testado e compartilhado!

A ideia do projeto PAC, era que, ao final do semestre o aluno tivesse condições para entregar um Robo seguidor de linha 100% funcional, a parte de componentes e a parte de codificação em C++.

A integração de matérias foi a parte fundamental para que pudessemos pegar o que aprendemos no PAC, na matéria de ferramentas WEB e também na matéria de fundamentos de programação para criar o robo seguidor de linha e também o WebSite com a apresentação do projeto.

ASSISTA O VÍDEO

Códigos dos projetos da Robótica



O código foi desensolvido utilizando a linguagem de programação C++ através da interface do arduino, optei por utilizar a biblioteca "QTRSensors.h" para um melhor desempenho do código, através da leitura dos Sensores QTR foi possível fazer o cálculo PID, que consiste no cálculo da velocidade e desaceleração do robô seguidor de linhas, onde a fórmula utilizada foi : Kp = 1;
Ki = 1;
Kd = 1;

s = tf('s');
C = Kp + Ki/s + Kd*s
que, explicando seria :
m(t) – será o resultado final, ou seja, a nossa saída do sistema.
Kc – também conhecida como ganho. É a variável que o sistema trabalhará, pode ver que ela está se multiplicando com o erro (e(t)) e como falamos a pouco o controlador proporcional trabalha no erro. É o Kc que vai proporcionar a abertura da válvula desde totalmente aberta à totalmente fechada. Tudo vai depender da coleguinha e(t).
e(t) (variável medida ) – Ela é a diferença entre o valor real e o valor desejado (setpoint).
